# 利用 Git Hook 对提交的代码进行检查

# 安装 GrumPHP

composer require --dev phpro/grumphp

安装完后会自动生成一个 grumphp.yml

请修改成：

parameters:
  git_dir: .
  bin_dir: vendor/bin
  tasks:
    phpcs:
      standard: PSR2
      ignore_patterns:
        - ./database/*
        - ./vendor/*

这样 git commit 的时候就会对提交的代码进行检查，只有符合 PSR-2 规范的代码才能通过。
